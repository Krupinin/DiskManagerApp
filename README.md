# DiskManagerApp

Приложение для работы с жесткими дисками. Треннировка разработки под Android. Позволяет добавлять, редактировать, удалять и фильтровать информацию о внутренних и внешних жестких дисках.

## Описание

DiskManagerApp - это Android-приложение, разработанное на Kotlin. Приложение предоставляет удобный интерфейс для управления коллекцией жестких дисков, включая возможность просмотра списка дисков, применения различных фильтров и выполнения операций CRUD (создание, чтение, обновление, удаление).

## Основные функции

- **Просмотр списка дисков**: Отображение всех сохраненных жестких дисков с их характеристиками
- **Фильтрация**: Возможность фильтрации по типу диска (все, высокая емкость, только внутренние, только внешние)
- **Статистика**: Отображение общей статистики по дискам (общее количество, отфильтрованное количество, количество внешних и внутренних дисков)
- **Добавление диска**: Форма для добавления нового жесткого диска с указанием типа, названия и емкости
- **Редактирование диска**: Возможность изменения характеристик существующего диска
- **Удаление диска**: Удаление выбранного диска из коллекции
- **Локализация**: Поддержка русского и английского языков

## Архитектура

Проект построен с использованием принципов Clean Architecture и разделен на модули:

- **app**: Основной модуль Android-приложения с пользовательским интерфейсом
- **core**: Бизнес-логика и доменная модель (entities, use cases, repository interfaces)
- **console**: Консольная версия приложения для демонстрации работы с данными

Архитектура внутри модулей:
- **MVVM** (Model-View-ViewModel) для разделения ответственности
- **Repository pattern** для абстракции доступа к данным
- **Dependency Injection** с использованием Hilt

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **База данных**: Room
- **Навигация**: Navigation Compose
- **Внедрение зависимостей**: Hilt
- **Архитектурные компоненты**: ViewModel, LiveData
- **Тестирование**: JUnit, Espresso, Compose Testing
